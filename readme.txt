วิธีใช้ 
	- เปิดไฟล์ ReadarChart (มันจะเปิดหน้าเว็บขึ้นมา)
	- ในหน้าเว็บจะแสดงกราฟให้เห็นด้านล่าง
	- ส่วน Tab[Setting] ด้านบนเมื่อกดแล้วจะมีให้ตั้งค่า
		- ChartTitle : ชื่อกราฟ
		- Name : ชื่อเส้นกำกับกราฟ
		- Name's Position : ตำแหน่งที่จะแสดง ชื่อเส้น
		- RGB : ใส่เลขสีของกราฟ
		- LabelList : เป็นกลุ่มค่าที่ใช้เป็นข้อมูลของกราฟ ปุ่มแดงไว้ลบค่านั้นออก ส่วนปุ่มเขียวไว้เพิ่มข้อมูลอีก
